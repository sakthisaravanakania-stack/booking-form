# Online Theater Booking Form

This project is a simple Online Theater Booking Form created using HTML and CSS. It was built to practice designing booking forms, collecting user input, and improving the user interface with basic CSS styling.

## Features

- Online theater ticket booking form
- Customer name input
- Email and phone number fields
- Movie selection
- Theater selection
- Show date and time
- Seat selection
- Number of tickets
- Booking button
- Clean and user-friendly layout

## Technologies Used

- HTML5
- CSS3
- Visual Studio Code
- Git
- GitHub

## CSS Concepts Practiced

- Borders
- Padding
- Margin
- Font Size
- Colors
- Button Styling
- Form Layout
- Input Field Styling

## Purpose

The purpose of this project is to strengthen my HTML and CSS fundamentals by building a simple online theater booking form and applying CSS styling techniques to create a clean and organized user interface.
